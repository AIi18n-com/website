# Introduction to AI i18n

### Providing AI Solutions for Localizing Multilingual Websites

In today's globalized society, cross-language communication has become an essential need. To conduct global business in this diverse world, we often provide multilingual versions of websites, allowing users to interact in their native language environment. However, providing multilingual versions of websites is not an easy task, and traditional localization requires a lot of time and effort for translation work. Moreover, website maintenance is an ongoing process that requires continuous updates and management of translated content. These tasks require manual intervention, which is inefficient and costly (approximately ¥200,000 OR $30,000).

We use LLM models like ChatGPT to automate the translation work for multilingual websites, greatly reducing translation costs and making website localization easier for you.

### Easy Integration with One Line of Code

Our translation system is easy to use, requiring only one line of code to integrate it into your project. Whether you use Shopify, WordPress, React, Vue, or Vite, our solution is compatible with all platforms, making it easy to localize your website quickly across various platforms.

[Quick Start →](/docs/quick-start)

### Advanced Search Engine Optimization (SEO)

Our intelligent context translation system not only provides accurate and natural translations but also has advanced search engine optimization (SEO) capabilities. This will help improve your website's search engine rankings, making it easier for you to stand out in overseas markets and attract more potential customers for your business.

### Excellent Translation Content Management System

Our translation system provides powerful backend management features that allow you to easily manage translated content. You can manage domain terms in the backend to ensure the professionalism and accuracy of the translated content. At the same time, we also support version control, allowing you to track the history of translation changes and ensure that the content always meets your needs.

### REST API Interface

Our intelligent context translation system provides flexible REST API interfaces, making it easy for you to manage translated content through APIs to meet various application scenarios. Whether you are a developer or a non-technical person, our API provides a simple and fast way to integrate, making website localization easier for you.
