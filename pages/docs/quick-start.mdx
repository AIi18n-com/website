# Quick Start

开始使用 AI i18n 很简单，应该不会超过几分钟。如果您还没有注册，请先按照说明注册并创建您的第一个网站/Web 应用程序项目。

## Step 1：安装代码片段

复制并粘贴以下代码片段到您的网站应用程序的 HTML 的 `<head>` 标签中。即可开始使用 AI i18n。

> 请将 `[PROJECT_ID]` 和 `[API_KEY]` 替换为您的项目 ID 和 API Key。可以在 AI i18n [管理后台的 **项目** ↗](https://app.aii18n.com/projects) 页面找到。

```html
<script
  id="aii18n"
  projectid="[PROJECT_ID]"
  apikey="[API_KEY]"
  src="https://unpkg.com/aii18n@^5.3.1"
></script>
```

## Step 2：验证安装是否成功

安装完代码片段后，刷新你的网站/应用程序。
AI i18n 会自动检测您的网站/应用程序的语言，并发送给 AI i18n 管理后台。

你可以通过以下方式验证安装是否成功：

- 在 AI i18n 管理后台的 **项目** 页面，查看是否有您的网站上报的词条，如果有，说明安装成功。
- 在控制台（Chrome/Firefox）中，查看是否有 `aii18n` 的日志输出，如果有看到 `aii18n: init success`，说明安装成功。
